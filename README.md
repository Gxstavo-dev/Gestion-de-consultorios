# MindDesk 🧠

Sistema de gestión de consultorio psicológico, diseñado para facilitar
el manejo de pacientes, citas, notas de sesión, seguimiento emocional y pagos.

## ¿Qué es MindDesk?

MindDesk es una plataforma opensource para psicólogos que permite gestionar
su consultorio de forma digital, segura y eficiente.

## Características

-  Gestión de pacientes
-  Agenda de citas
-  Notas de sesión privadas
-  Seguimiento emocional con gráficas
-  Control de pagos
-  Autenticación segura con httpOnly cookies
